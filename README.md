Quran Data Analysis

Overview: Data analysis on the Quran, exploring patterns and themes.

Features:

- Data preprocessing
- Visualization

Tech:

- Python 3.x
- Pandas, NumPy, Matplotlib, Seaborn, Wordcloud
- Jupyter Notebook

Getting Started:

1. Clone repo
2. Install above libraries 
3. Run Jupyter Notebook
